# Лабораторная работа №19
### Динамические структуры данных. Списки

Решить задачи 1-6 используя рекурсивные функции обработки списка.

1. Посчитать количество нулевых элементов в списке.

3. Удалить заданное количество элементов, после первого элемента с заданным значением. 
4. Вставить заданное значение пред каждым элементом с другим заданным значением.
5. Заполнить список значениями из файла, расположив их в порядке не убывания значений.
6. Разделить список на два списка, оставив в одном списке элементы с чётными номерами, а в другом с нечётными.

# Лабораторная работа №12
### С-cтроки

При решении данных заданий, использовать строки статического размера.

 1. Посчитать количество слов в строке (с кириллицей). 
 2. Найти в строке первое вхождение заданного символа.
 3. Найти в строке первое вхождение заданного слово.
 4. Удалить из строки каждое вхождение заданного символа.
 5. Вставить в строку другую строку после первого вхождения заданного символа. Если это невозможно, выдать сообщение об этом.

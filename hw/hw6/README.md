# Домашняя работа №6

Реализовать задания из Лабораторной работы №5 с использованием динамических массивов.

Каждая задача должна быть решена в одном из двух вариантах - с использованием разреженного либо уплотненного массива.

Для каждой задачи необходимо описать соответствующие тесты. При решении необходимо контролировать утечки памяти.

1. Удалить все нулевые столбцы из матрицы.

2. Добавить в матрицу строку с номером К. Вектор-строка передается как параметр.

3. Транспонировать не квадратную матрицу.

4. Найти произведение двух не квадратных матриц.
